## What is ShadowCode?

ShadowCode is a typing-based Python coding trainer where learners don't read code first ‚Äî they physically write it into muscle memory by typing over a transparent "ghost".

**Not tutorials. Not videos. Not copy-paste. Embodied coding.**

## How it works

1. **Ghost Code Layer**: Predefined Python code snippet appears faded and uneditable
2. **Live Typing Layer**: You type directly on top of the ghost code
3. **Accuracy Engine**: Tracks character accuracy and completion rate (speed is secondary)
4. **Progression System**: Start with simple statements, advance to complex code

## Current Features

- Beginner Python level with `print()` statement
- Real-time typing overlay
- Dark theme optimized for coding
- No copy-paste restrictions
- Auto-focus for immediate practice

## Getting Started

```bash

```

Open http://localhost:5175 and start typing the Python shadow!
